# 私用メモ: 準備中メイドさんを本掲載にする手順

プロフィール準備中で掲載しているメイドさんを、本掲載へ切り替える手順です。

## 1. PowerShellで作業場所へ移動

```powershell
cd "C:\Users\infop\OneDrive\ドキュメント\ポトロポータル\potoro-profile-public"
```

## 2. maids.jsonをメモ帳で開く

```powershell
notepad maids.json
```

## 3. 対象メイドさんを探す

例として `うさ` さんなら、次のような部分を探します。

```json
{
  "name": "うさ",
  "birthday": "?/?",
  "generation": "七代目",
  "likes": [
    "準備中"
  ],
  "favorite_food": "準備中",
  "one_liner": "プロフィール準備中です♡",
  "image": "img/profile preparation.png",
  "status": "preparing"
}
```

## 4. 本掲載の内容に変更する

主に変更する項目です。

- `birthday`: 実際の誕生日にする
- `likes`: 好きなものにする
- `favorite_food`: 好きな食べ物にする
- `one_liner`: ひとことにする
- `image`: 本人写真にする場合は画像パスを変更する
- `status`: `"status": "preparing"` の行を削除する

例:

```json
{
  "name": "うさ",
  "birthday": "8/8",
  "generation": "七代目",
  "likes": [
    "アニメ",
    "甘いもの"
  ],
  "favorite_food": "オムライス、プリン",
  "one_liner": "たくさんお話しに来てください♡",
  "image": "img/usa.jpg"
}
```

本人写真を使う場合は、先に画像を `img/` フォルダへ入れておきます。

## 5. 保存して閉じる

メモ帳で保存して閉じます。

## 6. 検証

```powershell
& 'C:\Users\infop\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe' scripts\validate-maids.js
```

成功すると、このように表示されます。

```text
maids.json validation passed (19 entries).
```

## 7. GitHubへ反映

```powershell
git status
git add maids.json
git commit -m "Publish maid profile"
git push origin main
```

GitHub Pagesの反映後、ページ上では「準備中」チップが外れて本掲載になります。
